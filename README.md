HTML Basics Practice Project :-

📌 Project Title:-
HTML Page Demonstrating Lists, Tables, Text Formatting

📖 Description :-
This project is a simple beginner-level HTML practice page. It demonstrates the use of fundamental HTML elements such as ordered lists, unordered lists, definition lists, abbreviations and tables (timetable). The goal of this project is to understand how different HTML tags work together to build a structured web page.

🧩 Features Used in This Project :-
- Ordered List (`<ol>`)
- Unordered List (`<ul>`)
- Definition List (`<dl>`, `<dt>`, `<dd>`)
- Abbreviation Tag (`<abbr>`)
- Table for Timetable (`<table>`, `<tr>`, `<th>`, `<td>`)
- Images (`<img>`)
- Basic HTML structure (`<html>`, `<head>`, `<body>`)
